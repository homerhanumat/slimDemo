# Slim

Slim is a minimal, clean and beautiful theme for [Hugo](http://gohugo.io/).

This repo is the surce code for a Slim demo-site.  The slim theme was modified a bit for use with the [**blogdown**](https://github.com/rstudio/blogdown) R-package.

The theme repo is at:

>[https://github.com/homerhanumat/hugo-theme-slim](https://github.com/homerhanumat/hugo-theme-slim)
